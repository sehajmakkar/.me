## Introduction

Taskly is a minimal TODO app built from scratch using HTML, CSS, and JavaScript with Express.js for the backend and JWT for authentication.


## Features

1. Sign In
2. Jwt Based Authentication
3. CRUD Operations
4. Pure Full Stack ( build from Scratch )

#### User Workflow

1. The user Signs In.
2. The user can add a task.
3. The user can delete a task.
4. The user can mark a task as completed.
4. The user can filter tasks by status (All, Active, Completed).



#### Tech Stack

- HTML
- CSS
- JavaScript
- Express.js
- JWT


## Open-Source and No Hidden Costs

Taskly is an open-source project, and we are committed to keeping it that way. The application does not connect to any servers or third-party services, ensuring privacy and transparency.
